# ADS: Astargon Domain System

Foundry VTT v13 build 351 icin Astargon yerlesim ve domain yonetim modu.

## Hizli kullanim

- Mod etkinken `K` tusu ADS penceresini acar.
- GM tum yerlesimleri gorur, olusturur, siler, oyuncu atar ve aylik turu isler.
- Oyuncular yalnizca kendilerine atanmis yerlesimleri gorur.
- Oyuncular izin verildiyse bina POP atamalarini, asker sayilarini ve Turn Note alanini gunceller.
- Aylik tur sadece GM Controls sekmesinden GM tarafindan islenir.

## Ilk test verisi

- Total POP: 309

- Growth: +3%, tahmini +9 POP

## Veri saklama

ADS verisi world-level Foundry setting olarak saklanir. Actor sheet veya DnD5e mekaniklerine bagli degildir.
